特色

提供两种实现，基于数组和AVL树。

除了使用函数作为条件，导致的无法比较对象之间的大小的操作之外，所有的操作的最优时间复杂度都是O(log n)。对于数组来说，我们使用二分搜索来查找；对于AVL树来说，其实也是类似于数组的二分搜索。

它们唯一不同的地方就在于，数组在频繁插入时的复杂度就很差了。但是通常来说，我们的程序并不会频繁的进行插入操作，所以我定义的默认类型是数组，而非AVL树。

动态规划 - 换根 DP

发表于2023-08-10|更新于2023-12-24|OI
|字数总计:2.7k|阅读时长:13分钟

简介

换根 DP 是解决当根节点变化时答案也发生变化的问题的题目，通常需要两遍 DFS，一遍先钦定一个点为根处理信息，第二遍是换根 DP。

[POI2008] STA-Station

给定一个 n≤106n\le 10^6n≤106 个点的树，请求出一个结点，使得以这个结点为根时，所有结点的深度之和最大。

题目链接：P3478。

当根节点通过边 (fa,u)(fa,u)(fa,u) 转移时：

  1. 原先 uuu 的子树中的任何信息不变。

  2. 现在 fafafa 也是 uuu 的子结点，所以 fafafa 除了 uuu 以外的子树所有深度都加一，反映到 sum(u)sum(u)sum(u) 上就是：

    sum(u)←sum(fa)−[sum(u)+sz(u)]+[n−sz(u)]sum(u)\leftarrow sum(fa)-[sum(u)+sz(u)]+[n-sz(u)] sum(u)←sum(fa)−[sum(u)+sz(u)]+[n−sz(u)]

    第一个中括号里是原本 uuu 对 fafafa 的贡献。

深度之和，大概是 n2n^2n2 级别，注意开 long long。

[CSP-S2019] 树的重心

对于 nnn 个点的树中每条边，求断掉后分裂出的两个子树的重心编号之和，n≤3×105n\le 3\times 10^5n≤3×105。

题目链接：P5666。

重心性质 #1

首先不看这个题，证明一个充要条件：点 uuu 是重心 ⇔\Leftrightarrow⇔ 删除 uuu 后构成的连通块大小的最大值 ≤n/2\le n/2≤n/2

为了方便理解，我们这里换根为 uuu，所以删除 uuu 后的所有连通块就是以 uuu 的子结点为根的子树。

  1. 充分性：假设 ∃v∈son(u),sz1(v)>n/2\exists v\in son(u),sz_1(v)>n/2∃v∈son(u),sz1​(v)>n/2，那么以 vvv 为根时，sz2(u)≤n/2sz_2(u)\le n/2sz2​(u)≤n/2，此时 uuu 一定不是重心。

  2. 必要性：假设 uuu 不是重心，满足 ∀v∈son(u),sz1(v)≤n/2\forall v\in son(u),sz_1(v)\le n/2∀v∈son(u),sz1​(v)≤n/2，那么换到 vvv 为根时，sz2(u)=n−sz1(v)≥n/2sz_2(u)=n-sz_1(v)\ge n/2sz2​(u)=n−sz1​(v)≥n/2。如果 sz1(v)=sz2(u)=n/2sz_1(v)=sz_2(u)=n/2sz1​(v)=sz2​(u)=n/2，那么 u,vu,vu,v 都是重心，否则 uuu 是重心，因为换到相邻的点不会更优。

    这里 sz1sz_1sz1​ 指换根前的大小，sz2sz_2sz2​ 指换根后的大小。

重心性质 #2

一棵树如果存在多个重心，设 uuu 是其中一个重心，设存在边 (u,v)(u,v)(u,v) 那么有：

  1. sz(v)≤n/2sz(v)\le n/2sz(v)≤n/2，这是重心的性质。
  2. n−sz(v)≥n/2n-sz(v)\ge n/2n−sz(v)≥n/2，这是不等式的性质。

重心不可能在 vvv 的子树中，因为向下走 n−szn-szn−sz 会更大，所以重心只可能是 vvv，也就是两个重心之间必然有边，而树上不存在环，所以重心最多只有两个。

当有两个重心时，还需要满足 n−sz(v)≤n/2n-sz(v)\le n/2n−sz(v)≤n/2，于是 sz(v)=n−sz(v)=n/2sz(v)=n-sz(v)=n/2sz(v)=n−sz(v)=n/2，还需要 nnn 为偶数。

重心性质 #3

以 uuu 为根时，重心一定在 uuu 所在重链上。如果 uuu 为重心显然成立，如果不为重心有 sz(son(v))>n/2sz(son(v))>n/2sz(son(v))>n/2，继续跳，最后一个满足 n−sz(u)≤n/2n-sz(u)\le n/2n−sz(u)≤n/2 的一定是重心，如果有两个那么就是它的父结点。

证明：如果不是重心或者第二重心在下面，那么一定存在重儿子满足 sz(v)≤n/2sz(v)\le n/2sz(v)≤n/2，那么换到 vvv 就会有 n−sz(v)≤n/2n-sz(v)\le n/2n−sz(v)≤n/2，这样就和 uuu 是最后一个满足条件的矛盾。

所以我们可以用倍增求出这最后一个点。

[CF685B] Kay and Snowflake

给定 nnn 个点的树和 qqq 次询问，每次询问一个点 uuu，求以 uuu 为根时重心的标号，若有多个输出任意一个。

题目链接：CF685B。

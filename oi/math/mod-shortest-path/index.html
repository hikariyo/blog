数学 - 同余最短路

发表于2023-10-27|更新于2023-11-04|OI
|字数总计:2.1k|阅读时长:9分钟

简介

同余最短路是用最短路的方式求解「用若干个整数凑出其它整数」这一问题，可以看做类似于背包问题的一种 DP，但是需要用到最短路求解。

它的思想比较神奇，设我们有的数列是 aaa，我们用 a1a_1a1​ 当模数，f(i)f(i)f(i) 表示满足 x mod a1=ix \bmod a_1= ixmoda1​=i 的最小的 xxx。假设我们已经知道整个 fff 数组了，那么判断一个数 kkk 能否被凑出来，就看 f(k mod a1)≤kf(k\bmod a_1) \le kf(kmoda1​)≤k，如果是的话用 f(k mod a1)f(k\bmod a_1)f(kmoda1​) 加上若干倍的 a1a_1a1​ 就一定能凑出来 kkk，反之凑不出来。

这个状态转移应该这么进行：

f(i)=min⁡i=1n{f((j+ai) mod a1)+ai}f(i)=\min_{i=1}^n \{f((j+a_i)\bmod a_1) + a_i\} f(i)=i=1minn​{f((j+ai​)moda1​)+ai​}

满足一个最短路的形式，所以可以用最短路求解。

[POI2003] Sums

我们给定一个整数集合 AAA。考虑一个非负整数集合 A′A'A′，所有属于 A′A'A′ 的集合的数 xxx 满足当且仅当能被表示成一些属于 AAA 的元素的和（数字可重复）。

比如，当 A={2,5,7}A = \{2,5,7\}A={2,5,7}，属于 A′A'A′ 的数为：000（000 个元素的和），222，444（2+22 + 22+2）和 121212（5+75 + 75+7 or 7+57 + 57+5 or 2+2+2+2+2+22 + 2 + 2 + 2 + 2 + 22+2+2+2+2+2）；但是元素 111 和 333 不属于 A′A'A′。

输入格式

第一行有一个整数 nnn，代表集合 AAA 的元素个数。接下来每行一个数 aia_iai​ 描述一个元素。A={a1,a2,...,an}A = \{a_1,a_2,...,a_n\}A={a1​,a2​,...,an​}。

接下来一个整数 kkk，然后每行一个整数，分别代表 b1,b2,...,bkb_1,b_2,...,b_kb1​,b2​,...,bk​。

输出格式

输出 kkk 行。如果 bib_ibi​ 属于 A′A'A′，第 iii 行打印 TAK，否则打印 NIE。

对于所有数据，1≤n≤5×1031 \le n \le 5 \times 10^31≤n≤5×103，1≤k≤1041 \le k \le 10^41≤k≤104，1≤a1<a2<...<an≤5×1041 \le a_1 < a_2 < ... < a_n \le 5 \times 10^41≤a1​<a2​<...<an​≤5×104，0≤bi≤1090 \le b_i \le 10^90≤bi​≤109。

题目链接：P8060。

[国家集训队] 墨墨的等式

墨墨突然对等式很感兴趣，他正在研究 ∑i=1naixi=b\sum_{i=1}^n a_ix_i=b∑i=1n​ai​xi​=b 存在非负整数解的条件，他要求你编写一个程序，给定 n,a1…n,l,rn, a_{1\dots n}, l, rn,a1…n​,l,r，求出有多少 b∈[l,r]b\in[l,r]b∈[l,r] 可以使等式存在非负整数解。

输入格式

第二行 nnn 个整数 a1…na_{1\dots n}a1…n​。

输出格式

对于 100%100\%100% 的数据，n≤12n \le 12n≤12，0≤ai≤5×1050 \le a_i \le 5\times 10^50≤ai​≤5×105，1≤l≤r≤10121 \le l \le r \le 10^{12}1≤l≤r≤1012。

题目链接：P2371。

首先把数列里的 000 全部扔掉，没用。然后，把这个问题转化成求 0∼l−1,0∼r0\sim l-1,0\sim r0∼l−1,0∼r 内有多少满足要求的数字，也就是我们只需要判断 0∼x0\sim x0∼x 中有多少个数字满足要求。

枚举 iii 表示模 a1a_1a1​ 后的结果是多少，然后就是判断 0∼x0\sim x0∼x 中有多少个数字满足 mod a1=i\text{mod }a_1 = imod a1​=i，显然是只有 dist(i)∼xdist(i)\sim xdist(i)∼x 中的数字是满足要求的，那么问题就转化成了求一个区间内满足 mod a1=i\text{mod }a_1= imod a1​=i 的数字有多少个。

首先讨论一个更一般化的问题，在 [L,R][L,R][L,R] 中 mod p=r\text{mod }p = rmod p=r 的数字有多少个？转化成满足 L≤kp+r≤RL\le kp+r\le RL≤kp+r≤R 的整数 kkk 有多少个，接下来我们只需要找到第一个 kkk 和最后一个 kkk 就可以了，即：

⌊R−rp⌋−⌈L−Rp⌉+1\lfloor\frac{R-r}{p}\rfloor - \lceil\frac{L-R}{p}\rceil+1 ⌊pR−r​⌋−⌈pL−R​⌉+1

回到本题，也可以套公式：

⌊x−ia1⌋−⌈dist(i)−ia1⌉+1\lfloor\frac{x-i}{a_1}\rfloor-\lceil\frac{dist(i)-i}{a_1}\rceil+1 ⌊a1​x−i​⌋−⌈a1​dist(i)−i​⌉+1

由于 dist(i)dist(i)dist(i) 的实际意义是 mod a1=i\text{mod }a_1=imod a1​=i 的最小数字，那么中间的式子一定整除，所以还可以进一步化简：

⌊x−ia1⌋−dist(i)−ia1+1=⌊x−dist(i)a1⌋+1\lfloor\frac{x-i}{a_1}\rfloor-\frac{dist(i)-i}{a_1}+1=\lfloor\frac{x-dist(i)}{a_1}\rfloor+1 ⌊a1​x−i​⌋−a1​dist(i)−i​+1=⌊a1​x−dist(i)​⌋+1

[ABC077D] Small Multiple

给定一个正整数 KKK。求 KKK 的正整数倍的数位累加和最小是多少。2≤K≤1052 \le K \le {10}^52≤K≤105；

题目链接：ABC077D。

由于求 KKK 的倍数，所以可以考虑求 mod K\text{mod }Kmod K 意义下的 000 数位累加和最小，设 f(x)f(x)f(x) 是 x mod Kx \bmod KxmodK 意义下 xxx 最小数位累加和。

首先 f(1)=1f(1)=1f(1)=1，然后考虑如何进行转移。由于十进制下所有的整数都可以通过 (+1)(+1)(×10)(+1)(+1)(+1)(\times 10)(+1)(+1)(+1)(×10)(+1) 这种方式转移，考虑 (×10)(\times 10)(×10) 对数位和的影响是 000，(+1)(+1)(+1) 对数位和的影响是 111，那么：

f(i+1)=min⁡{f(i)+1}f(10i)=min⁡{f(i)}f(i+1)=\min\{f(i)+1\}\\ f(10i)=\min\{f(i)\} f(i+1)=min{f(i)+1}f(10i)=min{f(i)}

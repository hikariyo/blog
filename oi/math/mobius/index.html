定理

莫比乌斯反演是一个反解的过程，第一个版本是枚举 nnn 的约数 ddd：

F(n)=∑d∣nf(d)⇒f(n)=∑d∣nμ(d)F(nd)F(n)=\sum_{d|n}f(d)\\ \Rightarrow f(n)=\sum_{d|n}\mu(d)F(\frac{n}{d}) F(n)=d∣n∑​f(d)⇒f(n)=d∣n∑​μ(d)F(dn​)

直接把定理的结论展开进行证明：

RHS=∑d∣nμ(d)F(nd)=∑d∣nμ(d)∑t∣(n/d)f(t)=∑t∣nf(t)∑d∣(n/t)μ(d)=∑t∣nf(t)S(nt)=f(n)=LHS\begin{aligned} \text{RHS}&=\sum_{d|n}\mu(d)F(\frac{n}{d})\\ &=\sum_{d|n}\mu(d)\sum_{t|(n/d)}f(t)\\ &=\sum_{t|n}f(t)\sum_{d|(n/t)}\mu(d)\\ &=\sum_{t|n}f(t)S(\frac{n}{t})\\ &=f(n)=\text{LHS}\\ \end{aligned} RHS​=d∣n∑​μ(d)F(dn​)=d∣n∑​μ(d)t∣(n/d)∑​f(t)=t∣n∑​f(t)d∣(n/t)∑​μ(d)=t∣n∑​f(t)S(tn​)=f(n)=LHS​

交换求和顺序时，观察到 d=1d=1d=1 时 ttt 可以取遍 nnn 的所有约数，那当枚举每一个 ttt 时，与之对应的 ddd 一定有 t∣(n/d)t|(n/d)t∣(n/d)，也就是 d∣(n/t)d|(n/t)d∣(n/t)，这个写成倍数然后代一下就能看出来。

Let d=kn,k∈N∗RHS=∑n∣dμ(dn)∑d∣tf(t)=∑n∣tf(t)∑k∣(t/n)μ(k)=∑n∣tf(t)S(tn)=f(n)=LHS\text{Let }d=kn,k\in \N^*\\ \begin{aligned} \text{RHS}&=\sum_{n|d}\mu(\frac{d}{n})\sum_{d|t}f(t)\\ &=\sum_{n|t}f(t)\sum_{k|(t/n)}\mu(k)\\ &=\sum_{n|t}f(t)S(\frac{t}{n})\\ &=f(n)=\text{LHS} \end{aligned} Let d=kn,k∈N∗RHS​=n∣d∑​μ(nd​)d∣t∑​f(t)=n∣t∑​f(t)k∣(t/n)∑​μ(k)=n∣t∑​f(t)S(nt​)=f(n)=LHS​

Problem b

对于给出的 n 个询问，每次求有多少个数对 (x,y)，满足 a≤x≤b，c≤y≤d，且 gcd(x,y)=k。

题目链接：AcWing 2702，P2522。

首先用前缀和的思想将其转化为求一个矩形内满足要求的个数。

F(n)=∑i=1a∑j=1b[n∣gcd⁡(i,j)]f(n)=∑i=1a∑j=1b[n=gcd⁡(i,j)]F(n)=∑n∣df(d)\begin{aligned} F(n)&=\sum_{i=1}^a\sum_{j=1}^b[n|\gcd(i,j)]\\ f(n)&=\sum_{i=1}^a\sum_{j=1}^b[n=\gcd(i,j)]\\ F(n)&=\sum_{n|d}f(d) \end{aligned} F(n)f(n)F(n)​=i=1∑a​j=1∑b​[n∣gcd(i,j)]=i=1∑a​j=1∑b​[n=gcd(i,j)]=n∣d∑​f(d)​

根据莫比乌斯反演，那么就有：

f(n)=∑n∣dμ(dn)F(d)=∑i∈N∗μ(i)⌊ain⌋⌊bin⌋\begin{aligned} f(n)&=\sum_{n|d}\mu(\frac{d}{n})F(d)\\ &=\sum_{i\in\N^*} \mu(i) \lfloor\frac{a}{in}\rfloor\lfloor\frac{b}{in}\rfloor \end{aligned} f(n)​=n∣d∑​μ(nd​)F(d)=i∈N∗∑​μ(i)⌊ina​⌋⌊inb​⌋​

这里 kkk 的上限就是当右侧两个整除结果出现 0 之前。

约数个数和

设 d(x)d(x)d(x) 为约数个数，给定 n,mn,mn,m 求 ∑i=1n∑j=1md(ij)\sum_{i=1}^n\sum_{j=1}^md(ij)∑i=1n​∑j=1m​d(ij)

题目链接：P3327，AcWing 1358。

引理：d(ij)=∑x∣i∑y∣j[gcd⁡(x,y)=1]d(ij)=\sum_{x|i}\sum_{y|j}[\gcd(x,y)=1]d(ij)=∑x∣i​∑y∣j​[gcd(x,y)=1]

证明：对于任意一个公共质因子 pip_ipi​，都有 x,yx, yx,y 的取值都有 (1,1),(pi,1),⋯ ,(piαi,1),(1,pi),⋯ ,(1,piβi)(1, 1),(p_i,1),\cdots,(p_i^{\alpha_i},1),(1,p_i),\cdots, (1,p_i^{\beta_i})(1,1),(pi​,1),⋯,(piαi​​,1),(1,pi​),⋯,(1,piβi​​)，因此最终有 (αi+βi+1)(\alpha_i+\beta_i+1)(αi​+βi​+1) 种取值，这和约数个数是等价的，如果不同时包含 pip_ipi​，对应次数置为 0 即可。

F(t)=∑i=1n∑j=1m∑x∣i∑y∣j[t∣gcd⁡(x,y)]f(t)=∑i=1n∑j=1m∑x∣i∑y∣j[t=gcd⁡(x,y)]F(t)=∑t∣df(d)f(t)=∑t∣dμ(dt)F(d)\begin{aligned} F(t)&=\sum_{i=1}^n\sum_{j=1}^m \sum_{x|i}\sum_{y|j}[t|\gcd(x,y)]\\ f(t)&=\sum_{i=1}^n\sum_{j=1}^m \sum_{x|i}\sum_{y|j}[t=\gcd(x,y)]\\ F(t)&=\sum_{t|d}f(d)\\ f(t)&=\sum_{t|d}\mu(\frac{d}{t})F(d) \end{aligned} F(t)f(t)F(t)f(t)​=i=1∑n​j=1∑m​x∣i∑​y∣j∑​[t∣gcd(x,y)]=i=1∑n​j=1∑m​x∣i∑​y∣j∑​[t=gcd(x,y)]=t∣d∑​f(d)=t∣d∑​μ(td​)F(d)​

对 F(t)F(t)F(t) 变形时，先考虑到 x,yx,yx,y 分别可以取遍 1∼n,1∼m1\sim n, 1\sim m1∼n,1∼m，并且内部式子和 i,ji,ji,j 无关，因此内部式子的值只看 i,ji,ji,j 中有多少 x,yx,yx,y 的倍数。

F(t)=∑x=1n∑y=1m⌊nx⌋⌊my⌋[t∣gcd⁡(x,y)]\begin{aligned} F(t)&=\sum_{x=1}^n\sum_{y=1}^m \lfloor\frac{n}{x}\rfloor\lfloor\frac{m}{y}\rfloor[t|\gcd(x,y)]\\ \end{aligned} F(t)​=x=1∑n​y=1∑m​⌊xn​⌋⌊ym​⌋[t∣gcd(x,y)]​

由于 t∣x,t∣yt|x, t|yt∣x,t∣y 所以只需要枚举 ttt 的倍数的那个系数，设 x′=x/t,y′=y/t,n′=n/t,m′=m/tx'=x/t, y'=y/t,n'=n/t,m'=m/tx′=x/t,y′=y/t,n′=n/t,m′=m/t，这里都是向下取整，那个符号就不写了，结合文章开头提到的取整公式，则有：

F(t)=∑x′=1n′∑y′=1m′⌊n′x′⌋⌊m′y′⌋=(∑x′=1n′⌊n′x′⌋)(∑y′=1m′⌊m′y′⌋)\begin{aligned} F(t)&=\sum_{x'=1}^{n'}\sum_{y'=1}^{m'} \lfloor\frac{n'}{x'}\rfloor\lfloor\frac{m'}{y'}\rfloor\\ &=(\sum_{x'=1}^{n'} \lfloor\frac{n'}{x'}\rfloor)(\sum_{y'=1}^{m'}\lfloor\frac{m'}{y'}\rfloor) \end{aligned} F(t)​=x′=1∑n′​y′=1∑m′​⌊x′n′​⌋⌊y′m′​⌋=(x′=1∑n′​⌊x′n′​⌋)(y′=1∑m′​⌊y′m′​⌋)​

本质上就是常数可以提到外面去。

h(n)=∑x=1n⌊nx⌋f(t)=∑t∣dμ(dt)F(d)=∑t∣dμ(dt)h(⌊nd⌋)h(⌊md⌋)=∑k=1min⁡(n′,m′)μ(k)h(⌊n′k⌋)h(⌊m′k⌋)\begin{aligned} h(n)&=\sum_{x=1}^n\lfloor\frac{n}{x}\rfloor\\ f(t)&=\sum_{t|d}\mu(\frac{d}{t})F(d)\\ &=\sum_{t|d}\mu(\frac{d}{t})h(\lfloor\frac{n}{d}\rfloor)h(\lfloor\frac{m}{d}\rfloor)\\ &=\sum_{k=1}^{\min(n',m')}\mu(k)h(\lfloor\frac{n'}{k}\rfloor)h(\lfloor\frac{m'}{k}\rfloor) \end{aligned} h(n)f(t)​=x=1∑n​⌊xn​⌋=t∣d∑​μ(td​)F(d)=t∣d∑​μ(td​)h(⌊dn​⌋)h(⌊dm​⌋)=k=1∑min(n′,m′)​μ(k)h(⌊kn′​⌋)h(⌊km′​⌋)​

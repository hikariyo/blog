数据结构 - 分块与莫队

发表于2023-12-30|更新于2024-01-12|OI
|字数总计:9.8k|阅读时长:53分钟

简介

莫队算法是利用平衡复杂度的思想来优化暴力算法，通常是通过均值不等式求出一个最优复杂度，我不知道本文归到哪一类，感觉和数据结构关系更大一点。

如果一个问题的暴力方式询问和修改复杂度差距很大，假设是求区间和的题，维护原数组是修改 O(1)O(1)O(1)，查询 O(n)O(n)O(n)。如果我们分块并且处理出每块答案，块长为 BBB，那么修改仍然是 O(1)O(1)O(1)，但是查询变成了 O(nB+B)O(\frac{n}{B}+B)O(Bn​+B)，均值不等式可以得到当 B=nB=\sqrt{n}B=n​ 时有最优复杂度单次 O(n)O(\sqrt n)O(n​)。

所以莫队算法也是分块，不过是对查询分块，如果一个问题知道 [l,r][l,r][l,r] 区间的答案可以 O(1)O(1)O(1) 转移到 [l,r+1],[l+1,r],[l−1,r],[l,r−1][l,r+1],[l+1,r],[l-1,r],[l,r-1][l,r+1],[l+1,r],[l−1,r],[l,r−1] 的话，就可以用莫队算法，对 lll 分块并对 rrr 升序排序，当块长为 BBB 时：

  1. lll 移动次数是每次询问都在块内移动 mBmBmB。
  2. rrr 移动次数是每次换块都在整个区间上移动 n2B\frac{n^2}{B}Bn2​

故当 B=nmB=\frac{n}{\sqrt m}B=m​n​ 时取到最小值 O(nm)O(n\sqrt m)O(nm​) 复杂度。

除了这种普通的莫队算法以外，如果带修改，可以将修改视为第三维坐标；还可以将树通过括号序压缩到一个线性表上从而支持树上操作；当添加/删除其中一者简单另一者难时，可以记录下每块的历史操作，当需要进行难的操作时直接撤销历史操作，将删除与添加互换。

∑i=1kci2\sum\limits_{i=1}^k c_i^2 i=1∑k​ci2​

其中 cic_ici​ 表示数字 iii 在 [l,r][l,r][l,r] 中的出现次数。

对于 100%100\%100% 的数据，1≤n,m,k≤5×1041\le n,m,k \le 5\times 10^41≤n,m,k≤5×104。

题目链接：P2709。

[国家集训队] 小 Z 的袜子

小 Z 把这 NNN 只袜子从 111 到 NNN 编号，然后从编号 LLL 到 RRR 的袜子中随机选出两只来穿。尽管小 Z 并不在意两只袜子是不是完整的一双，他却很在意袜子的颜色，毕竟穿两只不同色的袜子会很尴尬。

你的任务便是告诉小 Z，他有多大的概率抽到两只颜色相同的袜子。当然，小 Z 希望这个概率尽量高，所以他可能会询问多个 (L,R)(L,R)(L,R) 以方便自己选择。

然而数据中有 L=RL=RL=R 的情况，请特判这种情况，输出 0/1。

100%100\%100% 的数据中，N,M≤50000N,M \leq 50000N,M≤50000，1≤L<R≤N1 \leq L < R \leq N1≤L<R≤N，Ci≤NC_i \leq NCi​≤N。

题目链接：P1494。

设 cic_ici​ 为 iii 的出现次数，所以答案就是：

∑(ci2)(r−l+12)\frac{\sum \binom{c_i}{2}}{\binom{r-l+1}{2}} (2r−l+1​)∑(2ci​​)​

求个 gcd⁡\gcdgcd 化简一下就好了。

[国家集训队] 数颜色

墨墨购买了一套 NNN 支彩色画笔（其中有些颜色可能相同），摆成一排，你需要回答墨墨的提问。墨墨会向你发布如下指令：

  1. Q L RQ\ L\ RQ L R 代表询问你从第 LLL 支画笔到第 RRR 支画笔中共有几种不同颜色的画笔。

  2. R P CR\ P\ CR P C 把第 PPP 支画笔替换为颜色 CCC。

为了满足墨墨的要求，你知道你需要干什么了吗？

对于所有数据，n,m≤133333n,m \leq 133333n,m≤133333

所有的输入数据中出现的所有整数均大于等于 111 且不超过 10610^6106。

题目链接：P1903。

添加时间维度 ttt，设块长为 BBB，则有：

  1. 移动 lll 是 nBnBnB 次。
  2. 移动 rrr 是 n2B\frac{n^2}{B}Bn2​ 次。
  3. 移动 ttt 是 n3B2\frac{n^3}{B^2}B2n3​ 次。

取 B=n23B=n^{\frac{2}{3}}B=n32​ 有复杂度为 O(n53)O(n^{\frac{5}{3}})O(n35​)，能够接受。移动时间时，判断是否会影响当前区间维护的答案，如果是则更新。

[CF940F] Machine Learning

给定 nnn 个数 aia_iai​，mmm 次询问：

  1. 查询对 [l,r][l,r][l,r] 内每个数出现次数的 mexmexmex。
  2. 单点修改。

其中 n,m≤105n,m\le 10^5n,m≤105。

题目链接：CF940F。

若一个区间的 mex=pmex=pmex=p，一定有 sum=O(p2)sum=O(p^2)sum=O(p2)，也就是说区间的 mexmexmex 是区间和的根号级别，而区间出现次数之和是 O(n)O(n)O(n)，于是本题的 mexmexmex 为 O(n)O(\sqrt n)O(n​)，于是可以暴力。

树上莫队

Count on a tree II

给定 nnn 点的树，每个点有颜色，mmm 询问 a,ba,ba,b 路径上颜色的种类数量。1≤n≤4×104,1≤m≤1051\le n\le 4\times 10^4,1\le m\le 10^51≤n≤4×104,1≤m≤105。

题目链接：SP10707。

将树用括号序压到一个区间内，指每个点第一次访问和回溯时加入序列，长度为 2n2n2n，用 L(u),R(u)L(u),R(u)L(u),R(u) 表示一个点在括号序中的第一次出现下标和第二次出现下标。

不妨设 L(a)<L(b)L(a)\lt L(b)L(a)<L(b)，u=LCA(a,b)u=LCA(a,b)u=LCA(a,b)，那么分两种情况：

  1. u=au=au=a，即 aaa 是 bbb 的祖先，此时 [L(a),L(b)][L(a),L(b)][L(a),L(b)] 就是 a∼ba\sim ba∼b 的路径上所有点，并且它们仅出现一次。
  2. u≠au\ne au=a，那么 [R(a),L(b)][R(a),L(b)][R(a),L(b)] 上经过 a∼ba\sim ba∼b 路径上除 uuu 以外的所有点，并且不在路径上的点都出现了两次。

[WC2013] 糖果公园

给定 nnn 点树，每棵树上有一种糖果，共 mmm 种，第 iii 种糖果的美味指数是 ViV_iVi​，游客第 jjj 次尝第 iii 个糖果增加的权值是 Vi×WjV_i\times W_jVi​×Wj​，qqq 次询问：

  1. 修改 xxx 点的糖果种类为 yyy。
  2. 询问 x,yx,yx,y 权值。

数据范围：1≤n,m,q≤1051\le n,m,q\le 10^51≤n,m,q≤105。

题目链接：P4074。

路径上的权值是 ∑i=1m∑j=1cntiWj×Vi\sum_{i=1}^m \sum_{j=1}^{\text{cnt}_i}W_j\times V_i∑i=1m​∑j=1cnti​​Wj​×Vi​，增量比较好维护，所以可以用树上莫队解决。

回滚莫队

Mex

给定 nnn 个元素 a1∼ana_1\sim a_na1​∼an​，mmm 次询问，每次询问区间内最小没有出现过的自然数，1≤n,m≤2×105,0≤ai≤2×1051\le n,m\le 2\times 10^5,0\le a_i\le 2\times 10^51≤n,m≤2×105,0≤ai​≤2×105。

题目链接：P4137。

可以用主席树在线 O(nlog⁡n)O(n\log n)O(nlogn) 解决本题，但是也可以用回滚莫队 O(nn)O(n\sqrt n)O(nn​) 离线解决，这里只是练习一下莫队。

回滚莫队就是无法简单的维护增加，但是维护删除比较简单（或者相反），于是可以将 lll 指针每次回滚到块头，rrr 指针换块时滚到块尾。这里有种可持久化的感觉，但是我们只需要记录下来需要回滚的操作历史，回滚时直接撤销这些操作即可。

第一个块的左边界是 111，所以初始化成任意一个非 111 值，每次都回滚到当前块的左边界，当 lll 与当前左边界不同时代表换块，重新以 O(n)O(n)O(n) 的代价初始化，于是复杂度是 O(nn)O(n\sqrt n)O(nn​)。

[JOISC2014] 历史研究

给定 nnn 个元素 a1∼ana_1\sim a_na1​∼an​，规定 aia_iai​ 的权值是 cntai×ai\text{cnt}_{a_i}\times a_icntai​​×ai​，mmm 次询问 [l,r][l,r][l,r] 内最大权值。1≤n,m≤105,1≤ai≤1091\le n,m\le 10^5,1\le a_i\le 10^91≤n,m≤105,1≤ai​≤109。

题目链接：JOISC2014C。

这题是添加元素简单，删除比较困难，所以需要回滚莫队。

首先获取到当前根据 lll 分块的块边界 L,RL,RL,R 与询问 l,rl,rl,r 我们想要让它只添加，所以这样操作：

  1. 如果 r≤Rr\le Rr≤R 直接暴力，复杂度 O(n)O(\sqrt n)O(n​)。
  2. 如果 r>Rr\gt Rr>R 由于我们在同一块内按 rrr 升序排序，所以不断加入新的元素；对于 lll，每次将其回滚到 R+1R+1R+1 即可。

这样平均复杂度仍为 O(nn)O(n\sqrt n)O(nn​)。调试的时候可以把块长放小，一般都是换块的时候写错了。

最远间隔距离

给定 nnn 个元素 a1∼ana_1\sim a_na1​∼an​，规定在一个区间内 aia_iai​ 的权值是 rai−lair_{a_i}-l_{a_i}rai​​−lai​​，即它最后一次出现的坐标与第一次出现的坐标之差，mmm 次询问 [l,r][l,r][l,r] 内的最大权值。1≤n,m≤2×105,1≤a≤2×1091\le n,m\le 2\times 10^5,1\le a\le 2\times 10^91≤n,m≤2×105,1≤a≤2×109。

题目链接：P5906。

注意回滚的时候需要检查是否是本次操作的第一次更新，如果不是第一次就忽略掉，所以倒序枚举操作历史。

[JSOI2009] 面试的考验

求区间最接近且不相等的两数之差的绝对值。

对于 100%100\%100% 的数据，1≤n,q≤1051\le n,q\le10^51≤n,q≤105，1≤ai≤1091\le a_i\le10^91≤ai​≤109。

题目链接：P5926。

处理绝对值的时候，就是查询前驱和后继，如果使用莫队，那么 O(nn)O(n\sqrt n)O(nn​) 的复杂度已经比较极限了，应该选用一个常数小的数据结构查询，可以是树状数组，也可以是压位 Trie，这里使用压位 Trie 了，搭配离散化进一步降低常数。

添加一个数字并且更新答案是容易的，这个步骤与 P2234 基本一样，但是删除数字是困难的，所以考虑使用回滚莫队。总复杂度 O(nnlog⁡64n)O(n\sqrt n\log_{64}n)O(nn​log64​n)，略微卡了下常，速度很快。

二次离线莫队

第十四分块（前体）

给定长度为 nnn 的序列 aaa，mmm 次查询 [l,r][l,r][l,r] 中 popcount(ai⊕aj)=k\text{popcount}(a_i\oplus a_j)=kpopcount(ai​⊕aj​)=k 且 i<ji\lt ji<j 的 (i,j)(i,j)(i,j) 数量。1≤n,m<105,0≤ai<2141\le n,m\lt 10^5,0\le a_i\lt 2^{14}1≤n,m<105,0≤ai​<214。

逆序对

区间逆序对数量，要求 O(nn)O(n\sqrt n)O(nn​)，无法接受 O(nnlog⁡n)O(n\sqrt n\log n)O(nn​logn)，链接：P5047。

当右指针向右拓展的时候，对答案的贡献应当是 f(l,r,r+1)f(l,r,r+1)f(l,r,r+1)，拆成前缀和就是 f(r,r+1)−f(l−1,r+1)f(r,r+1)-f(l-1,r+1)f(r,r+1)−f(l−1,r+1)，同样地，f(r,r+1)f(r,r+1)f(r,r+1) 的预处理比较轻松，但是 f(l−1,r+1)f(l-1,r+1)f(l−1,r+1) 就无法轻松做到这一点了。

所以，我们想要一个 O(1)O(1)O(1) 查询，O(n)O(\sqrt n)O(n​) 更新的数据结构来维护，因为有 O(nn)O(n\sqrt n)O(nn​) 次查询和 O(n)O(n)O(n) 次更新，所以这个数据结构就是分块。

用 f(l,x)f(l,x)f(l,x) 表示 1∼l1\sim l1∼l 中大于 xxx 的数量，用 f−1(r,x)f^{-1}(r,x)f−1(r,x) 表示 r∼nr\sim nr∼n 中小于 xxx 的数量。

  • 右指针向右加入新元素 f(l−1):[r+1,R]f(l-1):[r+1,R]f(l−1):[r+1,R]
  • 右指针向左删除旧元素 f(l−1):[R+1,r]f(l-1):[R+1,r]f(l−1):[R+1,r]
  • 左指针向左加入新元素 f−1(r+1):[L,l−1]f^{-1}(r+1):[L,l-1]f−1(r+1):[L,l−1]
  • 左指针向右删除旧元素 f−1(r+1):[l,L−1]f^{-1}(r+1):[l,L-1]f−1(r+1):[l,L−1]

并且处理出 g(r+1)=f(r,r+1),g−1(l−1)=f−1(l,l−1)g(r+1)=f(r,r+1),g^{-1}(l-1)=f^{-1}(l,l-1)g(r+1)=f(r,r+1),g−1(l−1)=f−1(l,l−1) 与上题相同。

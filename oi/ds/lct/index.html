数据结构 - Link Cut Tree

发表于2023-12-24|更新于2023-12-25|OI
|字数总计:3.5k|阅读时长:17分钟

简介

LCT 利用 Splay 对树进行虚实链剖分，同一实链上的结点处在同一个 Splay 中，排序的关键字是深度。因此 Splay 的中序遍历就是树的从上到下的有序的树链。虚边是只记录父结点，父结点不记录子结点。介绍一下几个重要函数。

isroot(x)

判断 xxx 是否为根，只需要判断 fa(x)fa(x)fa(x) 是否含有 xxx 的指针即可。

rotate(x)

与普通 Splay 不同的地方在于它需要先判断是否为根。

splay(x)

进行 Splay 操作之前需要把一路上所有结点懒标记下放，操作之后 xxx 是 Splay 的根。

access(x)

打通 xxx 所在树中根到 xxx 的实链，同时把 xxx 下面都变成虚边，操作之后 xxx 是 Splay 的根。

makeroot(x)

将 xxx 设为当前树上的根，当 access(x) 后，xxx 所在 Splay 中最大的就是 xxx 了，所以翻转一下，操作之后 xxx 是 Splay 的根。

findroot(x)

其实就是找前驱，操作之后树的根是 Splay 的根。

split(x, y)

抽离出 x,yx,yx,y 这条链，操作之后 yyy 是 Splay 的根。

link(x,y)

加边，这里如果 x,yx,yx,y 不在同一个树中，建立 x→yx\to yx→y 的虚边。

cut(x,y)

删边，首先 xxx 是 yyy 所在树的根，其次 fa(y)fa(y)fa(y) 直接等于 xxx 并且 yyy 没有左子树，即没有比 yyy 更接近 xxx 的结点时才删掉。

模板

给定 nnn 个点以及每个点的权值，要你处理接下来的 mmm 个操作。
操作有四种，操作从 000 到 333 编号。点从 111 到 nnn 编号。

  • 0 x y 代表询问从 xxx 到 yyy 的路径上的点的权值的 xor\text{xor}xor 和。保证 xxx 到 yyy 是联通的。
  • 1 x y 代表连接 xxx 到 yyy，若 xxx 到 yyy 已经联通则无需连接。

数据范围：

  • 1≤n≤1051 \leq n \leq 10^51≤n≤105，1≤m≤3×1051 \leq m \leq 3 \times 10^51≤m≤3×105，1≤ai≤1091 \leq a_i \leq 10^91≤ai​≤109。
  • 对于操作 0,1,20, 1, 20,1,2，保证 1≤x,y≤n1 \leq x, y \leq n1≤x,y≤n。
  • 对于操作 333，保证 1≤x≤n1 \leq x \leq n1≤x≤n，1≤y≤1091 \leq y \leq 10^91≤y≤109。

题目链接：P3690。

[THUWC2017] 在美妙的数学王国中畅游

学渣小R 被大学的数学课程虐得生活不能自理，微积分的成绩曾是他在教室里上的课的最低分。然而他的某位陈姓室友却能轻松地在数学考试中得到满分。为了提升自己的数学课成绩，有一天晚上（在他睡觉的时候），他来到了数学王国。

数学王国中，每个人的智商可以用一个属于 [0,1][0,1][0,1] 的实数表示。数学王国中有 nnn 个城市，编号从 0∼n−10 \sim n-10∼n−1，这些城市由若干座魔法桥连接。每个城市的中心都有一个魔法球，每个魔法球中藏有一道数学题。

每个人在做完这道数学题之后都会得到一个在 [0,1][0,1][0,1] 内的分数。一道题可以用一个函数 f(x)f(x)f(x) 表示。若一个人的智商为 xxx，则他做完这道数学题之后会得到 f(x)f(x)f(x) 分。有三种形式：

  • 正弦函数 f(x)=sin⁡(ax+b) (a∈[0,1],b∈[0,π],a+b∈[0,π])f(x)=\sin(a x + b)\ (a \in [0,1], b \in [0,\pi],a+b\in[0,\pi])f(x)=sin(ax+b) (a∈[0,1],b∈[0,π],a+b∈[0,π])
  • 指数函数 f(x)=eax+b (a∈[−1,1],b∈[−2,0],a+b∈[−2,0])f(x)=\text e^{ax+b}\ (a\in [-1,1], b\in [-2,0], a+b\in [-2,0])f(x)=eax+b (a∈[−1,1],b∈[−2,0],a+b∈[−2,0])
  • 一次函数 f(x)=ax+b (a∈[−1,1],b∈[0,1],a+b∈[0,1])f(x) = ax + b\ (a\in [-1,1],b\in[0,1],a+b\in [0,1])f(x)=ax+b (a∈[−1,1],b∈[0,1],a+b∈[0,1])

数学王国中的魔法桥会发生变化，有时会有一座魔法桥消失，有时会有一座魔法桥出现。但在任意时刻，只存在至多一条连接任意两个城市的简单路径（即所有城市形成一个森林）。在初始情况下，数学王国中不存在任何的魔法桥。

数学王国的国王拉格朗日很乐意传授小R 数学知识，但前提是小R 要先回答国王的问题。这些问题具有相同的形式，即一个智商为 xxx 的人从城市 uuu 旅行到城市 vvv（即经过 u→vu \to vu→v 这条路径上的所有城市，包括 uuu 和 vvv）且做了所有城市内的数学题后，他所有得分的总和是多少。

输入格式

第一行两个正整数 n,mn,mn,m 和一个字符串 typetypetype。表示数学王国中共有 nnn 座城市，发生了 mmm 个事件，该数据的类型为 typetypetype。

typetypetype 字符串是为了能让大家更方便地获得部分分，你可能不需要用到这个输入。其具体含义在 【限制与约定】 中有解释。

接下来 nnn 行，第 iii 行表示初始情况下编号为 iii 的城市的魔法球中的函数。一个魔法用一个整数 fff 表示函数的类型，两个实数 a,ba,ba,b 表示函数的参数，若

  • f=1f=1f=1，则函数为 f(x)=sin⁡(ax+b)f(x)=\sin(ax+b)f(x)=sin(ax+b)
  • f=2f=2f=2，则函数为 f(x)=eax+bf(x)=\text e^{ax+b}f(x)=eax+b
  • f=3f=3f=3，则函数为 f(x)=ax+bf(x)=ax+bf(x)=ax+b

接下来 mmm 行，每行描述一个事件，事件分为四类。

  • appear u v 表示数学王国中出现了一条连接 uuu 和 vvv 这两座城市的魔法，保证连接前 u,vu,vu,v 这两座城市不能互相到达。

  • disappear u v 表示数学王国中连接 u,vu,vu,v 这两座城市的魔法桥消失了，保证这座魔法桥是存在的。

  • magic c f a b 表示城市 ccc 的魔法球中的魔法变成了类型为 fff，参数为 a,ba,ba,b 的函数

  • travel u v x 表示询问一个智商为 xxx 的人从城市 uuu 旅行到城市 vvv 后，他得分的总和是多少。若无法从 uuu 到达 vvv，则输出一行一个字符串 unreachable。

输出格式

对于每个询问，输出一行实数，表示得分的总和。

对于 100%100\%100% 的数据，1≤n≤105,1≤m≤2×1051\le n\le 10^5, 1\le m \le 2 \times 10^51≤n≤105,1≤m≤2×105 。

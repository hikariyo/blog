数据结构 - 并查集

发表于2023-06-14|更新于2023-11-14|OI
|字数总计:5.7k|阅读时长:26分钟

简介

该数据结构的作用有如下两点：

  1. 合并两个集合。
  2. 查询两个元素是否在同一个集合中。

基本实现原理是把每个集合用一颗树表示，树根的编号是这个集合的编号，每个节点都存储它的父节点，p[x] 表示 x 的父节点，对于根节点，p[x] = x。

对于优化，可以在查找树根时将路径上的每一个节点的父节点都指向树根。

例题

一共有 nnn 个数，编号是 111 ~ nnn，最开始每个数各自在一个集合中。现在要进行 mmm 个操作，操作共有两种：

  1. M a b，将编号为 aaa 和 bbb 的两个数所在的集合合并，如果两个数已经在同一个集合中，则忽略这个操作；
  2. Q a b，询问编号为 aaa 和 bbb 的两个数是否在同一个集合中；

[NOI2002] 银河英雄传说（带权并查集）

有一个划分为 N 列的星际战场，各列依次编号为 1,2,…,N。

有 N 艘战舰，也依次编号为 1,2,…,N，其中第 i 号战舰处于第 i 列。

有 T 条指令，每条指令格式为以下两种之一：

  1. M i j，表示让第 i 号战舰所在列的全部战舰保持原有顺序，接在第 j 号战舰所在列的尾部。
  2. C i j，表示询问第 i 号战舰与第 j 号战舰当前是否处于同一列中，如果在同一列中，它们之间间隔了多少艘战舰。

现在需要你编写一个程序，处理一系列的指令。

输入格式

第一行包含整数 T，表示共有 T 条指令。

接下来 T 行，每行一个指令，指令有两种形式：M i j 或 C i j。

其中 M 和 C 为大写字母表示指令类型，i 和 j 为整数，表示指令涉及的战舰编号。

输出格式

你的程序应当依次对输入的每一条指令进行分析和处理：

如果是 M i j 形式，则表示舰队排列发生了变化，你的程序要注意到这一点，但是不要输出任何信息；

如果是 C i j 形式，你的程序要输出一行，仅包含一个整数，表示在同一列上，第 i 号战舰与第 j 号战舰之间布置的战舰数目，如果第 i 号战舰与第 j 号战舰当前不在同一列上，则输出 −1。

题目链接：AcWing 238，P1196。洛谷后面有图解样例。

用前缀和的思想，设 did_idi​ 是 iii 到根节点的距离，那么两点间存在的点数应该是：

di,j={0,i=j∣di−dj∣−1,i≠jd_{i,j}=\left \{\begin{aligned} &0,i=j\\ &|d_i-d_j|-1,i\ne j \end{aligned}\right . di,j​={​0,i=j∣di​−dj​∣−1,i=j​

当合并两个集合 A,BA, BA,B 时，设将 BBB 合并到 AAA， 那么集合 BBB 中根节点的距离就要更新成 dB=∣A∣d_B=|A|dB​=∣A∣，然后它的子节点求距离的时候就可以找到正确的跳板求解了。

[NOI2001] 食物链

动物王国中有三类动物 A,B,C，这三类动物的食物链构成了有趣的环形。

A 吃 B，B 吃 C，C 吃 A。

现有 N 个动物，以 1∼N 编号。

每个动物都是 A,B,C 中的一种，但是我们并不知道它到底是哪一种。

有人用两种说法对这 N 个动物所构成的食物链关系进行描述：

第一种说法是 1 X Y，表示 X 和 Y 是同类。

第二种说法是 2 X Y，表示 X 吃 Y。

此人对 N 个动物，用上述两种说法，一句接一句地说出 K 句话，这 K 句话有的是真的，有的是假的。

当一句话满足下列三条之一时，这句话就是假话，否则就是真话。

  1. 当前的话与前面的某些真的话冲突，就是假话；
  2. 当前的话中 X 或 Y 比 N 大，就是假话；
  3. 当前的话表示 X 吃 X，就是假话。

你的任务是根据给定的 N 和 K 句话，输出假话的总数。

题目链接：AcWing 240，P2024。

结点 $u \to v $ 表示 uuu 被 vvv 吃，设某个点到根结点的距离为 ddd，判断 d mod 3d\bmod 3dmod3 的取值：

  • 余 0：当前点和根结底为同类。

  • 余 1：当前点吃根结点。

  • 余 2：当前点被根节点吃。

于是如果给的两个点 u,vu, vu,v 已经在集合中，就可以判断出它们的关系了。

当合并两个集合使 px→pypx \to pypx→py 的时候，如果 x,yx, yx,y 是同类，那么要保证：

如果 xxx 吃 yyy 那么要使得：

dx+dpx≡dy+1(mod 3)dpx≡dy−dx+1\begin{aligned} d_x+d_{px}&\equiv d_y + 1(\text{mod }3)\\ d_{px}&\equiv d_y-d_x+1 \end{aligned} dx​+dpx​dpx​​≡dy​+1(mod 3)≡dy​−dx​+1​

白雪皑皑

现在有 nnn 片雪花排成一列。 pty 要对雪花进行 mmm 次染色操作，第 iii 次染色操作中，把第 ((i×p+q) mod n)+1((i\times p+q)\bmod n)+1((i×p+q)modn)+1 片雪花和第 ((i×q+p) mod n)+1((i\times q+p)\bmod n)+1((i×q+p)modn)+1 片雪花之间的雪花（包括端点）染成颜色 iii。其中 p,qp,qp,q 是给定的两个正整数。他想知道最后 nnn 片雪花被染成了什么颜色。没有被染色输出 000。

输入格式

输入共四行，每行一个整数，分别为 n,m,p,qn,m,p,qn,m,p,q，意义如题中所述。

输出格式

输出共 nnn 行，每行一个整数，第 iii 行表示第 iii 片雪花的颜色。

数据范围

对于 100%100\%100% 的数据满足：1≤n≤1061\leq n\leq 10^61≤n≤106，1≤m≤1071\leq m\leq 10^71≤m≤107，1≤m×p+q,m×q+p≤2×1091\leq m\times p+q,m\times q+p\leq 2\times 10^91≤m×p+q,m×q+p≤2×109

线段树 85pts

看到区间修改首先想到了线段树，复杂度是 O((m+n)log⁡n)≈O(2×108)O((m+n)\log n)\approx O(2\times 10^8)O((m+n)logn)≈O(2×108)，加上常数很难过。

并查集 100pts

这是一道并查集维护连通性的题目，对于每个点，我们用 fa(i)≥ifa(i)\ge ifa(i)≥i 表示它不能染到的第一个点，由于染色是会被覆盖的，我们只要倒着染色就达到了跳过已经操作的元素的目的。

[CEOI1999] Parity Game

Alice 和 Bob 在玩一个游戏：他写一个由 000 和 111 组成的序列。Alice 选其中的一段（比如第 333 位到第 555 位），问他这段里面有奇数个 111 还是偶数个 111。Bob 回答你的问题，然后 Alice 继续问。Alice 要检查 Bob 的答案，指出在 Bob 的第几个回答一定有问题。有问题的意思就是存在一个 010101 序列满足这个回答前的所有回答，而且不存在序列满足这个回答前的所有回答及这个回答。

输入格式

第 111 行一个整数 nnn，是这个 010101 序列的长度。

第 222 行一个整数 mmm，是问题和答案的个数。

第 333 行开始是问题和答案，每行先有两个整数，表示你询问的段的开始位置和结束位置。然后是 Bob 的回答。odd表示有奇数个 111，even 表示有偶数个 111。

输出格式

输出一行，一个数 xxx，表示存在一个 010101 序列满足第 111 到第 xxx 个回答，但是不存在序列满足第 111 到第 x+1x+1x+1 个回答。如果所有回答都没问题，你就输出所有回答的个数。

对于 100%100\%100% 的数据，1≤n≤1091 \le n \leq 10^91≤n≤109，m≤5×103m \leq 5 \times 10^3m≤5×103。

题目链接：P5937。

首先发现序列长度远大于读入量，需要离散化。

然后我们可以通过前缀和的思想，每次读入一个 L,RL,RL,R 相当于知道了 SR−SL−1S_R-S_{L-1}SR​−SL−1​ 的奇偶性，也就是知道了 SRS_RSR​ 和 SL−1S_{L-1}SL−1​ 的奇偶性是否相同。这个用带权并查集维护即可，两个点如果不连通就连通起来，否则判断它们到根的距离是否模二同余。

[HAOI2006] 旅行

Z 小镇是一个景色宜人的地方，吸引来自各地的观光客来此旅游观光。Z 小镇附近共有 nnn 个景点（编号为 1,2,3,…,n1,2,3,\ldots,n1,2,3,…,n），这些景点被 mmm 条道路连接着，所有道路都是双向的，两个景点之间可能有多条道路。

也许是为了保护该地的旅游资源，Z 小镇有个奇怪的规定，就是对于一条给定的公路 rir_iri​，任何在该公路上行驶的车辆速度必须为 viv_ivi​。

速度变化太快使得游客们很不舒服，因此从一个景点前往另一个景点的时候，大家都希望选择行使过程中最大速度和最小速度的比尽可能小的路线，也就是所谓最舒适的路线。

输入格式

第一行包含两个正整数 n,mn,mn,m。

接下来的 mmm 行每行包含三个正整数 x,y,vx,y,vx,y,v。表示景点 xxx 到景点 yyy 之间有一条双向公路，车辆必须以速度 vvv 在该公路上行驶。

最后一行包含两个正整数 s,ts,ts,t，表示想知道从景点 sss 到景点 ttt 最大最小速度比最小的路径。sss 和 ttt 不可能相同。

输出格

如果景点 sss 到景点 ttt 没有路径，输出 IMPOSSIBLE。否则输出一个数，表示最小的速度比。如果需要，输出一个既约分数。

数据范围

对于 100%100\%100% 的数据，1≤x,y≤n≤5001 \le x,y \le n \le 5001≤x,y≤n≤500，1≤v<3×1041 \le v < 3 \times 10^41≤v<3×104，1≤m≤5×1031 \le m \le 5 \times 10^31≤m≤5×103，x≠yx \ne yx=y。

题目链接：P2502。

看到 mmm 比较小，所以想法是首先枚举固定一个最大值/最小值，这里选择固定最小值。

然后，想看看最大值到什么时候能使得 s,ts,ts,t 连通。

所以，先排序，然后枚举最小边，然后不断增量地找右边界，一旦找到了就记录一下答案，然后结束循环，剪纸。

染色

给定 nnn 点的树和 qqq 条路径，需要保证同一路径上的颜色相同，只能染成黑或白，求黑白点数颜色差的所有可能，n,q≤3×105n,q\le 3\times 10^5n,q≤3×105。

本题有两个部分，一个部分是处理出所有连通块的大小，另一部分是这些大小能够凑出的所有方案。

我在赛时对于第一部分使用了树剖，每次对一个路径上的点权用它们的 LCALCALCA 取 max⁡\maxmax，然而这个做法是错的，因为每次修改一条路径如果影响到了别的路径，是无法通过一个正确的复杂度去全部修改点权。

所以正解是使用并查集，每个点在并查集中的父亲代表它所在路径中最上层的 LCALCALCA。

对于第二部分，形式化的说，就是已知 ∑ai=n\sum a_i =n∑ai​=n 求 aia_iai​ 能凑出什么数字，显然是 01 背包问题。对于前一个条件，我们可以发现最多只存在 O(n)O(\sqrt n)O(n​) 个不同的 aia_iai​，分两种情况考虑：

  1. 当 ai≤na_i\le\sqrt nai​≤n​ 时，显然只有 n\sqrt nn​ 种。
  2. 当 ai>na_i\gt\sqrt nai​>n​时，如果超过了 n\sqrt nn​ 种，那么最终的总和就会大于 nnn。

综上，只有 O(n)O(\sqrt n)O(n​) 个不同的数字，所以我们先转化成多重背包问题，此时复杂度硬做还是 O(n×n×n)=O(n2)O(n\times \sqrt n\times \sqrt n)=O(n^2)O(n×n​×n​)=O(n2)，但是我们可以通过二进制拆分再次转化成 01 背包问题，会从 O(n)O(\sqrt n)O(n​) 个物品拆出 O(nlog⁡n)O(\sqrt n \log n)O(n​logn) 个物品，所以此时复杂度降到了 O(nnlog⁡n)O(n\sqrt n\log n)O(nn​logn)，用 bitset 加速可以再来一个小常数。

加边

给定 nnn 个点 mmm 条的带权无向图，给定序列 aaa，表示花费 ai+aja_i+a_jai​+aj​ 可以将边 (i,j)(i,j)(i,j) 的权修改为正无穷，qqq 个询问，每个询问代表将所有 ≥x\ge x≥x 的边连起来后，最小花多少代价可以将整个图连通。

不难发现如果保证 xxx 从大到小排列，那么每过一个询问都可以加入新的边，并且维护这些连通块，所以想到离线询问加上并查集维护。

我们要最小化连通块之间的 ∑ai+aj\sum a_i+a_j∑ai​+aj​，如果固定 aia_iai​，那么就变成了 (cnt−1)ai+(∑aj)−ai(cnt-1)a_i + (\sum a_j) - a_i(cnt−1)ai​+(∑aj​)−ai​，我们维护所有连通块的最小值 aia_iai​，然后维护所有连通块最小值之和 S=∑ajS=\sum a_jS=∑aj​，每个询问的答案就是 (cnt−2)ai+S(cnt-2)a_i + S(cnt−2)ai​+S。

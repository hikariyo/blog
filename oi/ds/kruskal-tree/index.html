数据结构 - Kruskal 重构树

简介

Kruskal 重构树是用一颗二叉树来记录下 Kruskal 算法中合并两点所在连通块这一步骤的顺序，也就是加边顺序，新建一个虚点并将左右代表点依次与虚点连接，从而记录下来一次加边操作，最后 LCA(a,b)LCA(a,b)LCA(a,b) 就记录 (a,b)(a,b)(a,b) 间第一次加边时的信息。

Peaks

给定一张 nnn 个点、mmm 条边的无向图，第 iii 个点的权值为 aia_iai​，边有边权。

有 qqq 组询问，每组询问给定三个整数 u,x,ku, x, ku,x,k，求从 uuu 开始只经过权值 ≤x\leq x≤x 的边所能到达的权值第 kkk 大的点的权值，如果不存在输出 −1-1−1。

本题强制在线。即：每次查询输入的是 u′,x′,k′u', x', k'u′,x′,k′，则 u=(u′xor⁡lastans) mod n+1u = (u' \operatorname{xor} \text{lastans}) \bmod n + 1u=(u′xorlastans)modn+1，kkk 的解密方式与之相同，x=x′xor⁡lastansx = x' \operatorname{xor} \text{lastans}x=x′xorlastans。

对于 100%100\%100% 的数据，1≤n≤1051 \leq n \leq 10^51≤n≤105，0≤m,q≤5×1050 \leq m, q \leq 5 \times 10^50≤m,q≤5×105，1≤s,t≤n1 \leq s, t \leq n1≤s,t≤n，1≤ai,w≤1091 \leq a_i, w \leq 10^91≤ai​,w≤109，0≤u′,x′,k′<2310 \leq u', x', k' < 2^{31}0≤u′,x′,k′<231。

建 Kruskal 重构树后，可以倍增找到 xxx 能到达的最高点，查询最高点对应的 [L,R][L,R][L,R] 内第 kkk 大，自然想到主席树。

[CF1706E] Qpwoeirut and Vertices

多测，给出 nnn 个点， mmm 条边的不带权连通无向图， qqq 次询问至少要加完编号前多少的边，才能使得 [l,r][l,r][l,r] 中的所有点两两连通。

数据范围：∑n≤105,∑m,q≤2×105\sum n\le10^5,\sum m,q\le 2\times10^5∑n≤105,∑m,q≤2×105。

题目链接：CF1706E。

建立 Kruskal 重构树，由于这里边权是编号，所以直接边读边建即可，要使得 [l,r][l,r][l,r] 所有点连通，就要使得 dfndfndfn 最小和最大的 [L,R][L,R][L,R] 连通，而恰好 LCA(L,R)LCA(L,R)LCA(L,R) 就是答案。

[CF1628E] Groceries in Meteor Town

给定 nnn 个点的树，起初每个节点都是黑色。给 qqq 个询问，三种类型：

  1. 把编号为 [l,r][l,r][l,r] 的点染成白色。
  2. 把编号为 [l,r][l,r][l,r] 的点染成黑色。
  3. 询问从结点 xxx 出发到达任意一个白色节点的简单路径上经过的边，最大可能的权值。不存在则输出 −1-1−1。

题目链接：CF1628E。

由于本身就是一棵树，所以路径唯一，询问最大边权，所以升序排序，这样 LCA(a,b)LCA(a,b)LCA(a,b) 表示的就是 a,ba,ba,b 间最长边权。

求的是所有白点中和 xxx 的 LCALCALCA 深度最浅的那个，可以先求出所有白点的 LCALCALCA，然后 LCA(u,x)LCA(u,x)LCA(u,x) 就是答案结点，因为一定存在一个白点 vvv 使得 LCA(v,x)=LCA(u,x)LCA(v,x)=LCA(u,x)LCA(v,x)=LCA(u,x)，这个可以想两两求 LCALCALCA 的过程，一定存在一个点它与别人的 LCALCALCA 本来就是公共 LCALCALCA，所以这个做法是正确的。

[IOI2018] 狼人

在日本的茨城县内共有 NNN 个城市和 MMM 条道路。这些城市是根据人口数量的升序排列的，依次编号为 000 到 N−1N - 1N−1。每条道路连接两个不同的城市，并且可以双向通行。由这些道路，你能从任意一个城市到另外任意一个城市。

你计划了 QQQ 个行程，这些行程分别编号为 000 至 Q−1Q - 1Q−1。第 i(0≤i≤Q−1)i(0 \leq i \leq Q - 1)i(0≤i≤Q−1) 个行程是从城市 SiS_iSi​ 到城市 EiE_iEi​。

狼人的生活并不容易。当你是人形时，你必须避开人少的城市，而当你是狼形时，你必须避开人多的城市。对于每一次行程 i(0≤i≤Q−1)i(0 \leq i \leq Q - 1)i(0≤i≤Q−1)，都有两个阈值 LiL_iLi​ 和 Ri(0≤Li≤Ri≤N−1)R_i(0 \leq L_i \leq R_i \leq N - 1)Ri​(0≤Li​≤Ri​≤N−1)，用以表示哪些城市必须要避开。准确地说，当你是人形时，你必须避开城市 0,1,…,Li−10, 1, \ldots , L_i - 10,1,…,Li​−1 ；而当你是狼形时，则必须避开城市 Ri+1,Ri+2,…,N−1R_i + 1, R_i + 2, \ldots , N - 1Ri​+1,Ri​+2,…,N−1。这就是说，在行程 iii 中，你必须在城市 Li,Li+1,…,RiL_i, L_i + 1, \ldots , R_iLi​,Li​+1,…,Ri​ 中的其中一个城市内变身。

你的任务是，对每一次行程，判定是否有可能在满足上述限制的前提下，由城市 SiS_iSi​ 走到城市 EiE_iEi​。你的路线可以有任意长度。

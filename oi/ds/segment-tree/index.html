数据结构 - 线段树

发表于2023-07-14|更新于2023-12-03|OI
|字数总计:10.7k|阅读时长:55分钟

简介

线段树是用于维护一个区间内具有结合律数据的高效数据结构，单次操作复杂度为 O(log⁡n)O(\log n)O(logn)，但常数较大。

将一个区间不断以 mid=⌊l+r2⌋,[l,mid],[mid+1,r]mid=\lfloor \cfrac{l+r}{2}\rfloor,[l,mid],[mid+1,r]mid=⌊2l+r​⌋,[l,mid],[mid+1,r] 为区间分为两个子树，由于除了最后一层所有层都是满的，所以用类似于存储堆的形式，即数组来存储线段树，一个结点编号为 nnn 那么左子结点是 2n2n2n，右子结点是 2n+12n+12n+1。

设倒数第二层有 2k2^k2k 个结点，那么上面有 ∑i=0k−12i=2k−1\sum_{i=0}^{k-1} 2^i = 2^k-1∑i=0k−1​2i=2k−1 个结点，下面一层最多有 2k+12^{k+1}2k+1 个结点，设总共有 NNN 个元素，坏的情况下应该是倒数第二层很多，最后一层很少，因此保险起见开 4N4N4N 的空间绝对够用。

区间最大公约数

给定一个长度为 N 的数列 A，以及 M 条指令，每条指令可能是以下两种之一：

  1. C l r d，表示把 A[l],A[l+1],…,A[r],都加上 d。

结论是，一个数列与它的差分数列最大公约数相同。

gcd⁡(a1,a2,⋯ ,an)=gcd⁡(a1,a2−a1,⋯ ,an−an−1)\gcd(a_1,a_2,\cdots,a_n)=\gcd(a_1,a_2-a_1,\cdots,a_n-a_{n-1}) gcd(a1​,a2​,⋯,an​)=gcd(a1​,a2​−a1​,⋯,an​−an−1​)

考虑左边的任意一个约数 ddd，它一定也是右边的约数；右边的任意一个约数 ddd 也一定是左边的约数：ddd 时 a1,a2−a1a_1, a_2-a_1a1​,a2​−a1​ 的约数一定也是 (a2−a1)+a1(a_2-a_1)+a_1(a2​−a1​)+a1​ 的约数。

但是，对于一个数列的区间 [L,R][L,R][L,R] 它的差分的 gcd 就不是整个 gcd 了，需要先求出来第一项。

用差分使得此问题转化为单点修改与单点查询。

对于一个合法的选取方案，它的花费为被选中的最长区间长度减去被选中的最短区间长度。

区间 [li,ri][l_i,r_i][li​,ri​] 的长度定义为 (ri−li)(r_i-l_i)(ri​−li​) ，即等于它的右端点的值减去左端点的值。

求所有合法方案中最小的花费。如果不存在合法的方案，输出 −1-1−1。

输入格式

第一行包含两个整数，分别代表 nnn 和 mmm。

第 222 到第 (n+1)(n + 1)(n+1) 行，每行两个整数表示一个区间，第 (i+1)(i + 1)(i+1) 行的整数 li,ril_i, r_ili​,ri​ 分别代表第 iii 个区间的左右端点。

输出格式

输出一行一个整数表示答案。

数据范围

对于全部的测试点，保证 1≤m≤n1 \leq m \leq n1≤m≤n，1≤n≤5×1051 \leq n \leq 5 \times 10^51≤n≤5×105，1≤m≤2×1051 \leq m \leq 2 \times 10^51≤m≤2×105，0≤li≤ri≤1090 \leq l_i \leq r_i \leq 10^90≤li​≤ri​≤109。

题目链接：P1712。

首先观察到区间的顺序不影响答案，所以先按照长度升序排序。

由于我们要使得极差最小，如果排序后的 [l,r][l,r][l,r] 内的区间满足要求，当 rrr 向右移动的时候，lll 向左一定是不比当前答案优，所以 l,rl,rl,r 都会向右移动。

我们只需要判断加入这些区间后存在点被覆盖了 mmm 次，并且删除 lll 区间后会使得覆盖 <m<m<m 次，那么在 [l,r][l,r][l,r] 中选中 mmm 个互不相交的区间就一定包含了 l,rl,rl,r 两个区间。

P.S. 上面说的 l,rl,rl,r 都代表的是排好序的区间的索引。
